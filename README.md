# 深信服防火墙日志查询工具 v2.1

一个高效的防火墙日志查询工具，支持快速索引、多字段查询、增量更新等功能。

---

## 🚀 快速开始

### 安装部署

```bash
# 1. 上传脚本到服务器
scp sangforfw_log.sh root@your-server:/data/sangfor_fw_log_chaxun/

# 2. 配置日志目录（编辑脚本中的LOG_DIR变量）
vi sangforfw_log.sh

# 3. 运行脚本
bash sangforfw_log.sh

# 4. 建立索引（首次使用）
选择：3. 重建索引（全量）

# 5. 安装快捷键
选择：5. 快捷键管理 → 1. 安装快捷键
source ~/.bashrc

# 6. 开始使用
fwlog
```

## ⚡ CDB加速（可选）

### 什么是CDB？
TinyCDB是一个快速的键值存储库，专为只读查询优化，可将查询速度提升**10倍**。

### 安装TinyCDB
```bash
# Debian/Ubuntu
sudo apt-get install tinycdb

# CentOS/RHEL
sudo yum install tinycdb

# 验证安装
cdb -h
```

### 使用CDB加速
```bash
# 1. 安装TinyCDB后，重建索引会自动生成CDB索引
fwlog
选择：3. 更新索引

# 2. 查询时指定日期即可自动使用CDB加速
选择：2. 高级查询 → 1. 单IP查询
输入IP: 192.168.1.100
输入日期: 2026-04-27  # 指定日期启用CDB加速

# 输出会显示：[CDB加速模式]
```

### 性能对比
| 场景 | 文本索引 | CDB索引 | 提升 |
|------|---------|---------|------|
| 单IP查询（指定日期） | 0.45秒 | 0.04秒 | **11倍** |
| 批量查询（5个IP） | 2.3秒 | 0.8秒 | **3倍** |
| 高频查询（100次/分） | CPU 80% | CPU 15% | **稳定** |

详细说明：[CDB集成指南](CDB_INTEGRATION.md)

## 💡 使用示例

### 命令行快速查询

```bash
# 查询IP
fwlog -t 192.168.1.100

# 查询IP+端口
fwlog -t 192.168.1.100:443

# 查询端口
fwlog -t :8080

# 查询IP+日期范围
fwlog -t 192.168.1.100 -d 2026-01-01:2026-01-31
```

### 交互式菜单

```bash
# 打开主菜单
fwlog
```

```
│   │   └── by_query/              # 组合查询结果
│   └── temp/                      # 临时文件
└── logs/                          # 日志文件目录（需配置）
```

## 🐛 故障排除

### 问题1：Permission denied
**解决**：脚本会自动修复权限，直接运行即可
```bash
bash sangforfw_log.sh
```

### 问题2：fwlog命令不存在
**解决**：重新加载配置或重新安装快捷键
```bash
source ~/.bashrc
# 或
fwlog → 5. 快捷键管理 → 1. 安装快捷键
```

### 问题3：查询结果不准确
**解决**：更新索引
```bash
fwlog → 4. 增量更新索引
```

### 问题4：索引损坏
**解决**：重建索引
```bash
fwlog → 3. 重建索引（全量）
```
