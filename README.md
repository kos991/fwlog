# 深信服防火墙日志 SOC 专业版 v2.5.1

基于 Go + DuckDB + Vue 3 构建的高性能、精致化防火墙日志分析与溯源平台。支持全离线环境运行。

---

## 🌟 核心特性 (v2.5.1)

### 1. 监控大屏 (Dashboard)
- **实时流量趋势**：展示 24 小时内的每小时流量波动。
- **协议分布占比**：直观展示 TCP/UDP/ICMP 等协议分布。
- **风险 IP 排行**：自动统计并排行活跃的源 IP 地址。
- **政务网监控**：实时统计特定网段（政务网）的流量占比。

### 2. 深度溯源 (Tracing)
- **多维下钻查询**：支持时间、IP、端口、协议、动作的组合过滤。
- **联动工作台**：点击大屏任意图表元素，自动进入该维度的溯源列表。
- **NAT 关联视图**：一眼看清内网 IP 与 NAT 转换后的映射关系。

### 3. 智能 IP 引擎
- **政务网自动识别**：内置识别 `172.18.0.0/17`, `172.28.128.0/19`, `2.0.0.0/8` 并打标。
- **离线 GeoIP**：支持加载 `.mmdb` 库，识别全球公网 IP 归属地。
- **手动精准矫正**：支持通过 CSV 文件手动覆盖特定 IP 的标签与位置。

### 4. 增量高性能索引
- **后台自动扫描**：每 60 秒自动发现新日志并增量入库。
- **极速检索**：DuckDB 驱动，百万级数据秒级聚合。

## 🚀 快速开始

### 1. 放置离线依赖 (内网环境必做)
由于是单文件分发，请确保将以下资源放入程序同级的 `assets/vendor` 目录（或在编译前放入源码目录）：
- `assets/vendor/vue/vue.global.prod.js`
- `assets/vendor/echarts/echarts.min.js`
- `assets/vendor/element-plus/index.full.js`
- `assets/vendor/element-plus/index.css`

### 2. 准备数据文件
将以下文件放入 `data/` 目录：
- `GeoLite2-City.mmdb` (可选：用于地理位置识别)
- `custom_ip_map.csv` (可选：用于手动矫正，格式：`IP,标签,位置`)

### 3. 运行
```bash
./fwlog_pro.exe
```
访问：`http://localhost:8080`
